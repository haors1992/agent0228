# 开工模板（复制即用）

请先读取以下文件并作为本次任务的唯一上下文来源：
- README.md
- IMPLEMENTATION_REPORT.md
- IMPLEMENTATION_PROGRESS.md

然后按这个顺序执行并输出：
1. 用不超过 3 条复述你对当前任务的理解。
2. 给出最小执行步骤（默认只改 1 到 2 个文件）。
3. 明确不做全仓扫描，只在必要文件范围内操作。
4. 在动手改代码前，先列出将要修改的文件和原因。

执行约束：
- 小范围改动优先，默认总改动预算不超过 120 行；超过前先说明原因并征求确认。
- 不执行破坏性命令，不回滚我已有改动。
- 改动后只做最小必要验证（如编译、测试或关键路径验证）。
