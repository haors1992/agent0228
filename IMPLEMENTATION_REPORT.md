### 数据流图

```
用户查询
   ↓
[语义搜索] ──→ EmbeddingService.embed(query)
   ↓                          ↓
KnowledgeBaseManager       TextVector (100D)
   ↓                          ↓
遍历所有文档向量 ←───────────┘
   ↓
计算余弦相似度
   ↓
按相似度排序
   ↓
返回 top-K 文档
   ↓
添加到系统提示
   ↓
LLM API
   ↓
强化响应
```

